Описание скрипта¶

Зачем нужен?

  • Умный трейлинг-стоп. Своевременное определение смены тренда, т.е. доминирующего участника рынка (продавец/покупатель), просто необходимо для реализации «умного» трейлинг-стопа открытой позиций. Не забывайте: войти проще, чем выйти!
  • Точки входа. Индикатор возможно использовать для поиска потенциальных точек входа при смене тренда, но с большой осторожностью, так как окончание одного тренда не всегда является началом другого.

Предупреждение: поиск точек входа посредством этого индикатора подходит только очень опытными трейдерами. Примеры использования индикатора.

Что показывает?

В первую очередь, главный элемент индикатора - это:

  • уровень смены тренда, т.е. ближайшие:
    • поддержка в тренде (зоне) покупателей (трейлинг-стоп для лонга);
    • сопротивление в тренде (зоне) продавцов (трейлинг-стоп для шорта).

Основные события индикатора - точки выхода из позиции, отображаются метками двух типов:

  • «тейкпрофит» - частичное закрытие из-за ослабления тренда;
  • «смена тренда» - полное закрытие по трейлинг-стопу (из-за пересечения рынком актуального уровня смены).

Также индикатор подсвечивает фон графика в трендовых зонах.
TIP: Настройками предусмотрена быстрая смена Пресетов оформления индикатора.

Как работает?

Используя объемный анализ рынка, индикатор определяет опорные ценовые уровни, пробой которых с наибольшей вероятностью влечёт отмену текущей тенденции (тренда), т.е. прекращение (или смену) зоны продавцов/покупателей.

Опорные уровни имеют разную "силу" и ранжируются аглоритмами скрипта, позволяя выделять не только основной уровень смены тренда, но также дополнительные - для промежуточных тейкпрофитов открытой позиции.

Настройки индикатора¶

Начало расчётов¶

Установка начальной "точки" расчёта индикатора - т.е. даты и времени, с которого будет отрисован индикатор на графике. Задавать начало можно двумя способами:

  1. Выбрав необходимую дату и время при помощи выпадающего календаря.
  2. Перемещая мышкой на графике вертикальную линию с меткой "Начало" .

NOTE: При смене начальной точки индикатор будет полностью пересчитан и перерисован.

Код верификации¶

В это поле нужно ввести код активации индикатора. В префиксе кода содержится дата окончания действия подписки. Например, код вида 05_30_ХХХХХХ будет действовать до полудня 30 числа 05 месяца, т.е. до 30 мая.

Быстрые развороты¶

По умолчанию смена тренда осуществляется закрытием свечи за уровнем. Если включить опцию "Быстрые развороты", то смена тренда будет по касанию актуального уровня, т.е. тенью свечи.

Вкладка "Стиль"¶

В этой вкладке можно гибко настроить графическое оформление индикатора (линии, метки, цвет фона и т.д.) привычным способом.

Оповещения (раздел дополняется)¶

Индикатор генерирует индивидуальные оповещения по шести событиям:

Конец зоны продаж: пересечение рынком шортового стопа (сопротивления). Полное закрытие шортов.
Конец зоны покупок: пересечение рынком лонгового стопа (поддержки). Полное закрытие лонгов.
Тейкпрофит шорта по тренду: частичное сокращение шортов из-за ослабления тренда.
Тейкпрофит лонга по тренду: частичное сокращение лонгов из-за ослабления тренда.
Трейл поддержки: ступенчатое смещение (подтягивание) уровня поддержки.
Трейл сопротивления: ступенчатое смещение (подтягивание) уровня сопротивления.
